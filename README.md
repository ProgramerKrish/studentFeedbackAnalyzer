# NLP Student Feedback Analyzer

## Problem
Analyzing large amounts of student feedback manually is time-consuming and inefficient.

## Solution
This project uses NLP techniques to automatically analyze feedback and extract insights like sentiment and key concerns.

## Tech Stack
- Python
- SBERT (Sentence Transformers)
- Basic Clustering

## Features
- Sentiment understanding
- Semantic similarity using embeddings
- Grouping similar feedback

## Future Scope
- Can be extended to Healthcare to analyze patient feedback
- Integration with web apps using Django/React
